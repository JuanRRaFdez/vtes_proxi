# Fuentes

## IMPORTANTE: Instalar Matrix Extra Bold

Para obtener el aspecto correcto de las cartas, necesitas la fuente **Matrix Extra Bold**.

### Opción 1: Descarga desde internet
1. Busca "Matrix Extra Bold font download" en Google
2. Descarga el archivo `MatrixExtraBold.ttf` o `Matrix-ExtraBold.ttf`
3. Cópialo a esta carpeta: `proxicreator/fonts/`
4. Renómbralo a `MatrixExtraBold.ttf` si tiene otro nombre

### Opción 2: Descargar desde DaFont o similar
- **DaFont**: https://www.dafont.com/ (busca "Matrix")
- **1001 Fonts**: https://www.1001fonts.com/ (busca "Matrix Bold")
- **Google Fonts** (busca fuentes similares como "Bebas Neue" si no encuentras Matrix)

### Opción 3: Usar una fuente similar
Si no encuentras Matrix Extra Bold, puedes usar fuentes similares bold/negrita:
- **Bebas Neue** (muy similar)
- **Impact**
- **Arial Black**
