# Símbolos de Clanes

Coloca aquí los archivos SVG de los símbolos de cada clan.

## Formato de archivos

Todos los archivos deben ser SVG y seguir este formato de nombre:
- `toreador.svg`
- `gangrel.svg`
- `ventrue.svg`
- `brujah.svg`
- `malkavian.svg`
- `nosferatu.svg`
- `tremere.svg`
- etc.

El nombre del archivo debe estar en minúsculas y sin espacios.

## Tamaño recomendado

Los símbolos se renderizarán a 60x60 píxeles en la carta final.
Se recomienda que los SVG tengan un viewBox cuadrado para mejor visualización.
